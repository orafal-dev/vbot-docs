## Quick start

1. Place your `.lua` files in the bot's user **Scripts** folder.
2. Core libraries from `Scripts/core` are loaded automatically — use **PascalCase** module names (`Self`, `Map`, `Module`, etc.).
3. Start from the [recommended script template](/docs/getting-started/template).
4. Look up any function in the [API Reference](/docs/api-reference).

## Core principles

- Scripts run in a **cooperative** runtime — long loops must yield via `wait(ms)` or `Module` schedulers.
- Use only **documented** PascalCase APIs. Treat undocumented globals as unavailable.
- Always handle **nil** returns when game state or bindings are missing.
- Never use removed aliases like `self.*`, `map.*`, `CaveBot`, or `_G.position`.
