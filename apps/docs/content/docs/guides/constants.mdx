The `lua_consts` core library exports shared constants used across all scripts. Always prefer these named constants over magic numbers.

## Movement

### MoveDirection

| Constant | Value |
| --- | --- |
| `MoveDirection.NORTH` | 0 |
| `MoveDirection.EAST` | 1 |
| `MoveDirection.SOUTH` | 2 |
| `MoveDirection.WEST` | 3 |
| `MoveDirection.NORTHEAST` | 5 |
| `MoveDirection.SOUTHEAST` | 6 |
| `MoveDirection.SOUTHWEST` | 7 |
| `MoveDirection.NORTHWEST` | 8 |
| `MoveDirection.INVALID` | 9 |

> Value **4 is not valid**. Use constants 5–8 for diagonals.

## Cavebot

`WalkerEvent` — walker event type constants for `Cavebot.RegisterEvent`.

## Features

`BotFeatureId` — numeric IDs for public features (`HEALER`, `TARGETING`, `MAGIC_SHOOTER`, `TIMER_ACTIONS`, …). Prefer these or feature name strings with `Engine.Features` / `Features`. Objects Dumper is reserved and blocked.
