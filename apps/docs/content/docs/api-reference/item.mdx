#### `Item.Use(itemId)`

Uses an Item directly by id, like pressing an in-game item hotkey.
No backpack needs to be open and no container index or slot is required.
Use this for direct-use items such as Magic Shield Potions.

**Parameters**

| Name | Type | Optional | Description |
| :-- | :-- | :--: | :-- |
| `itemId` | `integer` |  | Item id to use |

**Returns**

| Type | Description |
| :-- | :-- |
| `boolean` | dispatched Whether the action was accepted for dispatch |

## Notes

Internal helper functions in the core library are not part of the public scripting API.
Use only the functions listed above when writing user scripts.
Descriptions, parameters, and return values come from `---` Lua annotation blocks in the core source when available,
and from typed signatures in the public API appendix otherwise.
